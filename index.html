dylan's pythagorean triple algorithm

what is this?

This website generates Pythagorean triples using Euclid’s formula. A Pythagorean triple is a set of three numbers (a, b, c) that satisfy:

a² + b² = c²

Enter a side length to find all right triangles where that value appears as either a leg or the hypotenuse.

  • Integer mode – only whole-number triples
  • Decimal mode – scaled decimal values
  • Fraction mode – exact fractional results, input must be a fraction

This tool is meant for exploring number theory, scaling behavior, and the structure behind right triangles.
